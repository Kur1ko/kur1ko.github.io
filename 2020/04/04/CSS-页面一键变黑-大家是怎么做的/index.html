CSS-页面一键变黑,大家是怎么做的

发表于 2020-04-04 | 分类于 css
字数统计: 372 字 | 阅读时长 ≈ 1 分钟

不考虑浏览器前缀主要有以下几种:

  1. filter: grayscale(100%); 将图像转换撑灰度图像, 值在0~100之间,默认是0,100%则完全转为灰度
  2. filter: #808080; 兼容ie的做法
  3. filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1); progid…到Microsoft这一串都为前缀,BasicImage为调整颜色处理、图像旋转、不透明度。参数BasicImage(Grayscale=’x’, Xray=’x’, Mirror=’x’, Invert=’x’, Opcity=’x’, Rotation=’x’)也是兼容ie的做法
