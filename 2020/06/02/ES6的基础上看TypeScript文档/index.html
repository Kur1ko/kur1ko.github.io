特殊

元祖Tuple: 表示一个已知数量和类型的数组, 各元素类型不必相同

枚举类型

Any

表示可能为任何类型

Void

表示没有任何类型

Never

表示永不存在的值的类型.

never类型是任何类型的子类,可以赋值给任何类型. 且任何类型都无法赋值给never类型,即使是any

类型断言

当你在知道传入值的确切类型的时候, 可以用类型断言告诉编译器这个变量应该被视为什么类型.(相当于java的强制类型转换)
